# AI Cognition Persuader

## Description
AI认知说服话术技能。用于生成基于认知心理学原理的说服性沟通话术，帮助用户在各种场景下更有效地传达观点、影响决策。

## Triggers
- "生成说服话术"
- "认知说服"
- "如何说服"
- "persuade"
- "说服技巧"

## Instructions
当用户需要生成说服性话术时，按以下步骤执行：

1. **场景分析**
   - 识别沟通场景（商务谈判、产品推介、团队管理等）
   - 分析目标受众特征
   - 明确说服目标

2. **认知原理应用**
   - 互惠原理：先给予价值
   - 承诺一致：建立小承诺
   - 社会认同：引用案例和数据
   - 权威效应：展示专业性
   - 稀缺性：强调独特价值
   - 喜好原则：建立共鸣

3. **话术生成**
   - 开场：建立信任和兴趣
   - 展开：逐步引导认知
   - 处理异议：预判并化解疑虑
   - 促成：推动决策行动

4. **输出格式**
   - 提供完整话术脚本
   - 标注关键认知原理
   - 给出使用建议和注意事项

## Examples

### Example 1: 产品推介
用户："帮我生成一个向企业客户推介AI工具的说服话术"

输出：
- 分析企业客户痛点
- 应用社会认同（同行案例）和稀缺性（限时优惠）
- 生成分阶段话术脚本

### Example 2: 团队管理
用户："如何说服团队接受新的工作流程"

输出：
- 识别变革阻力来源
- 应用承诺一致和参与感原理
- 提供渐进式沟通策略

## Notes
- 话术应真诚，避免操纵性语言
- 根据文化背景调整沟通风格
- 强调双赢而非单方面说服
- 提供多个话术版本供选择
