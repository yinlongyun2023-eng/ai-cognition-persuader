# AI Cognition Persuader

AI认知说服话术生成技能，基于认知心理学原理帮助用户创建有效的说服性沟通内容。

## 功能特点

- 🧠 **认知原理驱动**：基于6大经典说服原理（互惠、承诺一致、社会认同、权威、稀缺、喜好）
- 💬 **场景化话术**：支持商务谈判、产品推介、团队管理等多种场景
- 📝 **结构化输出**：提供完整的开场-展开-异议处理-促成话术脚本
- ✨ **真诚导向**：强调双赢沟通，避免操纵性语言

## 使用方法

### 在 Claude Code 中使用

将此 skill 放置在 `.claude/skills/` 目录下，然后在对话中使用触发词：

```
"帮我生成一个向企业客户推介AI工具的说服话术"
"如何说服团队接受新的工作流程"
"生成商务谈判的开场话术"
```

### 触发词

- 生成说服话术
- 认知说服
- 如何说服
- persuade

## 应用场景

1. **商务谈判**：价格谈判、合作协议、资源争取
2. **产品推介**：B2B销售、方案演示、客户说服
3. **团队管理**：变革推动、目标对齐、绩效沟通
4. **个人发展**：职业晋升、资源申请、跨部门协作

## 认知原理说明

### 1. 互惠原理
先给予价值，激发对方回报意愿

### 2. 承诺一致
从小承诺开始，逐步引导到大决策

### 3. 社会认同
引用案例、数据、同行实践建立信任

### 4. 权威效应
展示专业性、资质、成功经验

### 5. 稀缺性
强调独特价值、限时机会、资源有限

### 6. 喜好原则
建立共鸣、展现相似性、真诚赞美
